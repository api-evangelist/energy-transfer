# Energy Transfer (energy-transfer)
Energy Transfer LP operates one of the largest and most diversified portfolios of energy assets in the United States, with a strategic footprint in major production basins and transportation corridors. Energy Transfer provides the Messenger+ developer platform for pipeline messaging and gas scheduling API integrations.

## APIs

### Messenger+ API
The Energy Transfer Messenger+ API provides programmatic access to Energy Transfer's pipeline messaging and gas scheduling platform. The API enables partners and shippers to manage gas nominations, scheduling, and pipeline capacity through automated integrations, significantly reducing manual processing times.
